# Docks

The base Lineux design uses Dockslide, which provides the most optimized setup for running Lineux on your printer.

If you are using Dockslide, or if you are using fixed docks on a flying gantry printer, you may use the R2T dock files.

If you are not using Dockslide, or if Lineux is installed on a non-flying gantry printer where the docks are mounted directly to the printer frame, you will need to use the fixed dock files that match your specific printer carriage.

Docks for other toolheads can be found in their respective toolhead folders.
