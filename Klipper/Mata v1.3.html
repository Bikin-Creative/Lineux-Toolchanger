✅ How to integrate Mata Offsets

1. Download mata_offsets.cfg and place it in your Klipper config folder.
Add the following line to your printer.cfg:

[include mata_offsets.cfg]

2. Edit each tool_x.cfg file:

Replace the following block inside [gcode_macro _Variables_tX]:

variable_xoffset: 0
variable_yoffset: 0
variable_zoffset: 0

with this (depending on tool number):

ToolCode to insert
T0variable_xoffset: 0
variable_yoffset: 0
variable_zoffset: 0
T1variable_xoffset: {printer["gcode_macro mata_offsets"].t1_x | float}
variable_yoffset: {printer["gcode_macro mata_offsets"].t1_y | float}
variable_zoffset: {printer["gcode_macro mata_offsets"].t1_z | float}
T2variable_xoffset: {printer["gcode_macro mata_offsets"].t2_x | float}
variable_yoffset: {printer["gcode_macro mata_offsets"].t2_y | float}
variable_zoffset: {printer["gcode_macro mata_offsets"].t2_z | float}
T3variable_xoffset: {printer["gcode_macro mata_offsets"].t3_x | float}
variable_yoffset: {printer["gcode_macro mata_offsets"].t3_y | float}
variable_zoffset: {printer["gcode_macro mata_offsets"].t3_z | float}

Repeat for all tools (T1 to T7). T0 stays at 0.
